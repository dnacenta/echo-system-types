# pulse-system-types

Shared contract types for the [pulse-null](https://github.com/dnacenta/pulse-null) plugin ecosystem.

Defines the `Plugin` trait, LLM message types, monitoring types, and tool definitions that all pulse-null components depend on.

## Usage

```toml
[dependencies]
pulse-system-types = "0.5"
```
